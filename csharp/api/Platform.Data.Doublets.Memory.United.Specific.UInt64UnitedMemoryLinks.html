    Class UInt64UnitedMemoryLinks

    Represents a low-level implementation of direct access to resizable memory, for organizing the storage of links with addresses represented as UInt64.

    Представляет низкоуровневую реализация прямого доступа к памяти с переменным размером, для организации хранения связей с адресами представленными в виде UInt64.
